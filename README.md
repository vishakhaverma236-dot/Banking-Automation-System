# Banking Automation System

## Project Status
🚧 This project is currently under development.

## Description
A Banking Automation System built using Python, Tkinter, and MySQL. The application aims to automate common banking operations through a user-friendly graphical interface.

## Technologies Used
- Python
- Tkinter
- MySQL

## Planned Features
- User Login
- Customer Registration
- Deposit & Withdrawal
- Fund Transfer
- Transaction History
- Balance Enquiry
